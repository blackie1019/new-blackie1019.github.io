Detecting Duplicate Value in Column

接下來用countif這個函式來幫我判斷特定欄位是否有重複的值:

=countif(C4:C41,C4)>1

這邊當放上個寫法後就會去抓指定欄位是否有大於1的累加值，如果答案有重複的則會顯示TRUE，如果沒有重複則會顯示FALSE
